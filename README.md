# BSC DEX Arbitrage

Requirements:

Node 14 
Yarn 

Install:

```shell
yarn
```

Generate a mnemonic

```shell
npx hardhat generate
npx hardhat account
```

Never share the mnemonic.txt or the address text file with anyone
Back both of the .txt files up, securely, twice

Deploy

```shell
npx hardhat deploy --network bsc_testnet
```
